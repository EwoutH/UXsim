UXsim’s technical reference

UXsim is an open-source macroscopic and mesoscopic network traffic flow simulator developed in Python.

This webpage is a technical reference of UXsim codes, and may be too detailed. For basic explanation, please see the following pages and materials first:

  • Github repo

  • README

  • Tutorial based on Jupyter Notebook

  • arXiv preprint

For the summary of classes, please click the left-side panel.

Please note that this page is automatically generated from docstring of uxsim.py, and some of the docstrings are automatically written by ChatGPT. Thus, they are not completely perfect.
