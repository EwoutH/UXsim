UXsim's technical reference
===========================

UXsim is an open-source macroscopic and mesoscopic network traffic flow simulator developed in Python. 

This webpage is a technical reference of UXsim codes, and may be too detailed.
For basic explanation, please see the following pages and materials first:

* `Github repo <https://github.com/toruseo/UXsim>`_
* `README <https://github.com/toruseo/UXsim/blob/main/README.en.md>`_
* `Tutorial based on Jupyter Notebook <https://github.com/toruseo/UXsim/blob/main/demos_and_examples/demo_notebook_01en.ipynb>`_
* `arXiv preprint <https://arxiv.org/pdf/2309.17114.pdf>`_

For the summary of classes, please click the left-side panel.

Thus, they are not completely perfect.
