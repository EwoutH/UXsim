UXsim technical documentation
=============================

About
-----

UXsim is an open-source macroscopic and mesoscopic network traffic flow simulator developed in Python. 
This webpage is a technical documentation of UXsim codes.
Please visit `Github repo <https://github.com/toruseo/UXsim>`_ first.
